# 선지출 코드 체계

## 요약
- 형식: `KS-브랜드코드-번호` (예: KS-WB-01)
- 브랜드코드: WB(우방), DN(다뉴브), IT, N(공용/기타), VM(자판기), SV(매장소모품)
- 비용구분: KS(수리/교체), SV(매장소모품)

## 현재 문제점
- 번호 불연속 (중간 결번 다수)
- 브랜드코드 불일치 (WB 부품이 KS-N-XX에 등록)
- 중복 등록 (하이드림 70~72 vs 75~77)

## 제안
- 번호 연속 정리
- 브랜드코드 맞추기 (WB 부품 → KS-WB-XX로 이동)
- 중복 통합
