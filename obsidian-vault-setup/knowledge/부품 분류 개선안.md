# 부품 분류 개선안

## 요약
"공용"이라는 단어가 3가지 맥락에서 혼용되어 혼란 발생.

## 현재 문제
| 필드 | 값 | 의미 |
|------|------|------|
| brand | 공용 | 특정 브랜드 아닌 매장 물품 |
| group | 공용 | 1단/2단 구분 없이 전 기종 |
| device | (빈칸) | 세탁기/건조기 다 씀 |

## 제안 후보

**group "공용" 대체:**
- 기종호환 — 의미 정확, 직관적
- 공통 — 적응 쉬움, 혼동 가능
- 전기종 — 짧고 명확

**brand "공용" 대체:**
- 공동설비 — 여러 매장 공유 느낌
- 매장 — 짧고 UI 깔끔
- 매장설비 — 정확하지만 길다
