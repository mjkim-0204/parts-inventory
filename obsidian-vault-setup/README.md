# 옵시디언 Vault 세팅

## 사용법
이 폴더 내용물 전체를 `C:\Users\minni\Documents\Obsidian Vault\`에 복사하면 끝.

## 폴더 구조
```
_templates/     ← 메모 템플릿 (daily, project, knowledge)
daily/          ← 일일 메모 (자동 생성됨)
projects/       ← 프로젝트별 메모
knowledge/      ← 축적되는 지식/정보
ideas/          ← 아이디어 메모
```

## 옵시디언에서 해야 할 것 (3분)

1. 설정(⚙️) → 커뮤니티 플러그인 → 제한 모드 끄기
2. "찾아보기" → 아래 3개 검색 → 설치 → 활성화:
   - `Calendar`
   - `Dataview`
   - `Templater`
3. 설정 → Templater → Template folder: `_templates`
4. 설정 → 코어 플러그인 → Daily Notes 켜기
   - Date format: `YYYY-MM-DD`
   - New file location: `daily`
   - Template file: `_templates/daily`

끝! Ctrl+P → "Open daily note"로 바로 시작.
